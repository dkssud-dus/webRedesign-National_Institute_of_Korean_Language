## 1. 프로젝트 개요

### 1.1. 작업 배경
국립국어원은 국어와 발전과 국민의 언어 생활 향상을 연구 사업을 추진하고 체계적인 정책 수립의 기반을 마련하는 기관이다.


</br>
</br>
</br>

### 1.2. 키워드

1. **보다 쉬운**   
  1.1. 넓게 편듯한 레이아웃으로 여백 추가
  1.2. 사용자의 콘텐츠 사용도·이해 고려한 배치

</br>

2. **신뢰 가는**   
  2.1. 메인 비주얼의 강조
  2.2. 각 섹션별 목적의 시각화

</br>

3. **참여 지향**   
  3.1. 정보 제공뿐만 아니라, 사용자가 만드는 이벤트에 대한 반응 효과 추가
